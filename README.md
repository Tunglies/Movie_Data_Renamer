# Movie_Data_Renamer

本地**艺术电影**文件代码名称提取，配合电影元数据 抓起|刮削器使用以达到**艺术电影**文件名称格式化和本地流媒体服务器如 Emby，Jellyfin，Kodi，Plex 等入库管理。

# 使用

1. 复制 renamer.py 至待处理文件夹中终端运行

   ```bash
   python renamer.py
   ```
2. 复制 renamer.exe 至待处理文件夹中运行
